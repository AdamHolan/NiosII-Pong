# NiosII-Pong
One player pong for the Altera DE0 FPGA board.

CONTROLS:


Player 1:\
W to move up \
S to move down

Player 2: 
Not implemented

You can try this code using cpulator. Click on the UART terminal to initiate controls.  
https://cpulator.01xz.net/?sys=nios-de0
